Improving Customer experience on e-commerce website using Review Mining
=====================================================================================
This project is carried out as a part of Data Mining SWENG 545 , 2020 By Vrushab, Vineeth and Abhishek.

For more How to run the code:

1>Download the dataset from https://s3.amazonaws.com/amazon-reviews-pds/tsv/amazon_reviews_us_Electronics_v1_00.tsv.gz.

2>First, Run the fake reivew detection file followed by sentiment analysis.

3>Finally run CompleteModel.ipynb file cell by cell:
--a. final dataset after fake review will be used.
--b. Topic modelling.
--c. Sentiment Analysis will be done on topics.
--d. Text Summarization on Positive and Negative Reviews.

To download dataset, go to the link below:
https://s3.amazonaws.com/amazon-reviews-pds/readme.html
